# Mahir Uslu Portfolio

Personal portfolio rebuilt with Next.js App Router, TypeScript, Tailwind CSS,
and Vercel as the deployment target.

## Commands

- `bun install`: install dependencies and update `bun.lock`.
- `bun run dev`: start the local Next.js development server with Turbopack.
- `bun run build`: create a production build.
- `bun run lint`: run ESLint.
- `bun test`: run focused test coverage for data cleanup, blog fallback,
  and WhatsApp contact URL formatting.

The project targets Node `22.x` for deployment and uses Bun `1.3.9` as the
package manager.
